# Data for the causalnet tutorial (Applied Example)

This folder contains data used to reproduce the Applied Example in the causalnet tutorial manuscript.

Source and citation:
- Schumacher L, Klein JP, Elsaesser M, Härter M, Hautzinger M, Schramm E, Kriston L. Implications of the Network Theory for the Treatment of Mental Disorders: A Secondary Analysis of a Randomized Clinical Trial. JAMA Psychiatry. 2023 Aug 23:e232823. doi: 10.1001/jamapsychiatry.2023.2823.

- Schramm E, Kriston L, Zobel I, Bailer J, Wambach K, Backenstrass M, Klein JP, Schoepf D, Schnell K, Gumz A, Bausch P, Fangmeier T, Meister R, Berger M, Hautzinger M, Härter M. Effect of Disorder-Specific vs Nonspecific Psychotherapy for Chronic Depression: A Randomized Clinical Trial. JAMA Psychiatry. 2017 Mar 1;74(3):233-242. doi: 10.1001/jamapsychiatry.2016.3880. PMID: 28146251.
